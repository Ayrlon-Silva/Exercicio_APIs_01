API RandomFox

Descrição

A API RandomFox fornece imagens aleatórias de raposas de forma simples e gratuita. Ao fazer uma requisição HTTP, ela retorna um objeto JSON contendo a URL de uma imagem de raposa e um link de referência.

Demosntração via Curl

Comando: curl -L https://randomfox.ca/floof

Exemplo de saída esperada:


        {
            "image":"https:\/\/randomfox.ca\/images\/37.jpg",
            "link":"https:\/\/randomfox.ca\/?i=37"
        }
